Setup instructions for the the mui-tanstack react dev environment

Download Node
https://nodejs.org/en/download/
node-v16.17.0-x64
>npm install -g npm

Download dependencies
>npm install

"6 high severity warnings" might be ignorable, if Google is to believed

Ensure "REACT_APP_DATA_URL" is provided,
  or create one in a .env if it isn't.
  Ensure it has a trailing /
